if statement

In order to write useful programs, we almost always need the ability to check conditions and change the behavior of the program accordingly. Conditional statements give us this ability. The simplest form is the if statement:
myvar = 2 myothervar = 4 if myvar > 0 : print ('myvar is positive') if myvar == 2 : print ('myvar is 2') if myvar < 0: print ('myvar is negative') if myvar > 0: print ('myvar is positive') #notice both lines indented the same print ('So I printed 2 things this time.') #so both will print in this block if myvar < 0: print ('myvar is negative') print ('But if I do not get my indentation right, I might get weird execution.')
Just press Run!


if then else statement

A second form of the if statement is alternative execution, in which there are two possibilities and the condition determines which one gets executed. The syntax looks like this:
myvar = 2 myothervar = 4 if myvar > 0 : print ('myvar is positive') else : print ('myvar is 0 or negative') if myvar == 2 : print ('myvar is 2') else : print ('myvar is no equal to 2') if myothervar % 2 == 0: #modulo operation % - returns the remaider of the division print ('If the remainder of myvar is divided by 2 is 0, it is even') else: print('odd number!')
Just press Run!
Notice: again just keep in mind that your indentation (Whitespace) of the code blocks should be consistent.

Chained conditionals

Sometimes there are more than two possibilities and we need more than two branches. One way to express a computation like that is a chained conditional:
myvar = 2 if myvar < 0 : print ('myvar is negative') elif myvar == 0 : print ('myvar is 0') elif myvar == 10000000 : print ('myvar is a very big number') print ('but I know it is not big.') else : print('myvar is positive') #no conditional is true so this executes.
Just press Run!
elif is an abbreviation of "else if." Again, exactly one branch will be executed.
